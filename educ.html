Java HelloWorld Program

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, World!"); } }

public class HelloWorld { ... }: Defines a class named HelloWorld.

public static void main(String[] args) { ... }: Entry point of the application.

System.out.println("Hello, World!");: Prints "Hello, World!" to the console.

Java Variables and Data Types

public class VariablesExample { public static void main(String[] args) { int age = 25; double salary = 55000.50; String name = "Alice"; System.out.println("Name: " + name); System.out.println("Age: " + age); System.out.println("Salary: " + salary); } }

int age = 25; Declares an integer variable age.

double salary = 55000.50; Declares a double variable salary.

String name = "Alice"; Declares a string variable name.

System.out.println("Name: " + name); Prints the value of name.

Java Control Flow (If-Else Statement)

public class ControlFlowExample { public static void main(String[] args) { int number = 10; if (number > 0) { System.out.println("The number is positive."); } else if (number < 0) { System.out.println("The number is negative."); } else { System.out.println("The number is zero."); } } }

if (number > 0) { ... }: Checks if number is greater than 0.

else if (number < 0) { ... }: Checks if number is less than 0.

else { ... }: Executes if number is 0.

Java Loops (For Loop)

public class LoopExample { public static void main(String[] args) { for (int i = 0; i < 5; i++) { System.out.println("Iteration: " + i); } } }

for (int i = 0; i < 5; i++) { ... }: A loop that iterates 5 times.

System.out.println("Iteration: " + i); Prints the value of i.

Java Functions (Methods)

public class MethodExample { public static void main(String[] args) { int result = addNumbers(5, 7); System.out.println("Sum: " + result); } public static int addNumbers(int a, int b) { return a + b; } }

public static int addNumbers(int a, int b) { ... }: Defines a method to add two numbers.

return a + b; Returns the sum of a and b.

int result = addNumbers(5, 7); Calls the method with arguments 5 and 7.

int main() { ... }: Defines the main function where execution starts.

return 0; Indicates successful execution.
